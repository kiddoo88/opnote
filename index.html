บันทึก template: ข้อความในช่อง Procedure จะถูกจำไว้ในเครื่อง — เลือก operation เดิมครั้งหน้าจะได้ template ที่แก้แล้ว (มี ★ ในลิสต์) • ถ้าเลือก Other แล้วพิมพ์ชื่อ operation ใหม่ก่อนบันทึก จะเพิ่มเป็น operation ใหม่ในลิสต์เลย
Export PDF: กดปุ่มแล้วในหน้าต่างพิมพ์เลือกเครื่องพิมพ์เป็น "Save as PDF / บันทึกเป็น PDF" — ได้ไฟล์ตามแบบฟอร์ม IPD-OR-019 • ไฟล์ .doc เปิดแก้ต่อใน Word ได้ • ข้อมูลบันทึกแบบร่างในเครื่องอัตโนมัติ
⚙️ บันทึกข้อมูลลง Google Sheet (ตั้งค่าครั้งเดียว)
วิธีตั้งค่า: สร้าง Google Sheet ใหม่ → เมนู Extensions → Apps Script → วางโค้ดจากไฟล์ opnote_sheet_webapp.gs → Deploy → New deployment → Web app (Execute as: Me, Who has access: Anyone) → คัดลอก URL มาวางที่นี่ • เมื่อกด Export ทุกครั้ง ข้อมูลจะถูกเพิ่มเป็นแถวใหม่ใน Sheet อัตโนมัติ (ต้องต่ออินเทอร์เน็ต) • เว้นว่างไว้ = ไม่บันทึกลง Sheet
